## Shell Scripts

| Script | Description |
|--------|-------------|
| system_info.sh | Displays system information, kernel version, user details, uptime |
| package_inspector.sh | Checks if Linux Kernel is installed and displays package info |
| disk_audit.sh | Audits disk usage and permissions of system directories |
| log_analyzer.sh | Analyzes log files for specific keywords |
| manifesto_gen.sh | Generates personalized open source philosophy statement |

## How to Run

### Prerequisites
- Linux system (Ubuntu/CentOS/RHEL)
- Bash shell

### Steps
1. Clone the repository:
git clone https://github.com/meerashajii/oss-audit-24bcy10006.git
cd oss-audit-24bcy10006

2. Make scripts executable:
chmod +x *.sh

3. Run each script:
./system_info.sh
./package_inspector.sh
./disk_audit.sh
./log_analyzer.sh /var/log/syslog
./manifesto_gen.sh
